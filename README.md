# 📊 Customer Behavior Sales Analysis

## 📌 Project Overview

This project focuses on analyzing customer behavior and sales data to generate meaningful business insights. It demonstrates a complete data analysis workflow, starting from raw data cleaning to building an interactive dashboard for decision-making.

The project uses Python (Pandas) for data preprocessing, MySQL for querying and extracting insights, and Power BI for creating visualizations.

---

## 🎯 Objectives

* Understand customer purchasing behavior
* Identify top-selling products
* Analyze sales trends over time
* Support data-driven business decisions

---

## 🛠️ Technologies Used

* Python (Pandas)
* MySQL
* Power BI

---

## 🔄 Project Workflow

### Data Cleaning (Pandas)

* Handled missing values
* Removed duplicates
* Standardized data formats

### Data Analysis (MySQL)

* Wrote SQL queries to extract insights
* Identified important business metrics
* Analyzed customer and sales patterns

### Data Visualization (Power BI)

* Created an interactive dashboard
* Visualized key performance indicators (KPIs)
* Enabled easy exploration of insights

---

## 📊 Key Insights

* Identified high-value customers
* Discovered top-performing products
* Analyzed overall sales trends
* Improved understanding of customer behavior

## 🚀 How to Run the Project

1. Clone this repository
2. Run the Python script to clean the data
3. Import the cleaned data into MySQL
4. Execute SQL queries for analysis
5. Open the Power BI file to view the dashboard

The Power BI dashboard provides an interactive view of customer behavior and sales insights, helping users quickly understand trends and patterns.
